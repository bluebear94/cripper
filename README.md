## cripper

the C program that rips your PH3 archives apart

### Usage:

    # to compile it:
    make
    # to rip apart an archive:
    ./cripper --verbose archive.dat path/to/destination
    # same, but doesn't produce output:
    ./cripper archive.dat path/to/destination
